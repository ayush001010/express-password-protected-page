Secrets

  • Find All Saved WiFi Passwords: Use netsh wlan show profiles to list all saved networks, then netsh wlan show profile "WiFi-name" key=clear to view their passwords.
  • Check If Someone Is Using Your WiFi: Run arp -a to see all connected devices on your network.
  • Schedule Automatic Shutdown: Use shutdown -s -t 3600 to shut down after an hour, and shutdown -a to cancel it.
  • Create an Undeletable Folder: Run mkdir C:\con\ to create a system-protected folder and rd /s /q C:\con\ to remove it.
  • Search for a Word in a File: Use find "word" filename.txt to locate a specific word inside a text file.
  • Hide a File Inside Another: Use copy /b file1.txt + file2.txt newfile.txt to hide one file inside another.
  • Hide a File Inside a Folder: Use attrib +h foldername to hide a folder and attrib -h foldername to unhide it.
  • Hide a File Inside an Image: Run copy /b image.jpg + secret.txt hidden.jpg to merge a text file into an image for hidden storage.
  • Check System Information: Use systeminfo to get detailed information about your system.
  • Use wmic product get name to display all installed software on your PC.
  • Use tasklist to see all active tasks and taskkill /IM processname.exe to close a process.
  • Run tasklist to see all active tasks and taskkill /IM processname.exe /F to forcefully close a process.
  • Use ipconfig /all to display network details, including IP and MAC addresses.
  • Simply type speedtest (if speedtest-cli is installed) to test your internet speed.
  • Use ping google.com to check your internet connection and latency.
  • Run netstat -a to see all active connections and listening ports.
  • Use tracert google.com to trace the route packets take to reach a destination.
  • Run nslookup google.com to find the IP address of a domain.
  • Use whois domain.com to get registration details of a domain.
